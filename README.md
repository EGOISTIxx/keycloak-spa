# React + Keycloak SPA

## Конфигурация

Создать .env

```env
VITE_KEYCLOAK_URL=http://localhost:8080
VITE_KEYCLOAK_REALM=demo
VITE_KEYCLOAK_CLIENT_ID=keycloak-spa
```

## Run

```bash
git clone https://github.com/EGOISTIxx/keycloak-spa.git
cd keycloak-spa
npm install
npm run dev
```

## Keycloak

URL: ваш собственный URL

Realm: ваш собственный realm

ClientId: ваш собственный react-spa
